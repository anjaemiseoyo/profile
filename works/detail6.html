Breaking Mood, 2023,
웹, 설치.


Breaking Mood(분위기 깨기)는 상황에 어울리지 않는 소리를 재생해주는 웹 시스템이다. 분위기의 전환이 필요할때 웹 사이트의 돌을 누르면 상황에 맞지 않는 랜덤 소리가 재생된다. 소리 재생기는 돌탑의 형상으로 불안정하다. 버튼을 누르면 돌은 회전하면서 무너질 듯 무너지지 않는데 이는 곧 깨질듯 깨지지 않는 분위기와 연결된다. 실제 웹 링크를 다수의 사람에게 미리 공지하지 않은 채 배부하여 사이트 이용 후 느낀 반응을 설문데이터로 기록하였다. 설치 작업에서는 웹사이트의 비물질적 요소를 실제 돌 형태로 엮고 각각에 텍스트를 부여하여 존재감을 부여했다. 이 프로젝트는 작은 움직임이 결과적으로 조직의 구조적 변화의 불씨로 이어질 수 있는지에 관한 실험적인 작업이었다.


링크는 여기(모바일만 가능)

Breaking Mood is a web system that plays sounds that are not appropriate for the situation. When you need to change the atmosphere, press the stone on the website to play a random sound that is not appropriate for the situation. The sound reproducer is unstable in the shape of a stone tower. When the button is pressed, the stone rotates and does not collapse, which is connected to the atmosphere that does not break as if it were breaking. Actual web links were distributed to a large number of people without prior notice, and the reactions felt after using the site were recorded as questionnaire data. In the installation work, non-material elements of the website were woven into actual stones and text was given to each to give a sense of existence. This project was an experimental work on whether small movements could eventually lead to organizational structural changes.
